Detailed Description

Mapping type generated from the specification. Convertible with QMap<uint, SimplePresence>, but needed to have a discrete type in the Qt4 type system.

Mapping returned by GetPresences and signalled by PresencesChanged, indicating the presence of a number of contacts.
