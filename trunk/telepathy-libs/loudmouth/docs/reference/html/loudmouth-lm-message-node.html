Description

Represents a node in the LmMessage tree. These map to XML nodes in the XMPP protocol.

LmMessageNode

typedef struct {
	gchar      *name;
	gchar      *value;
	gboolean    raw_mode;

        LmMessageNode     *next;
        LmMessageNode     *prev;
	LmMessageNode     *parent;
        LmMessageNode     *children;
} LmMessageNode;
